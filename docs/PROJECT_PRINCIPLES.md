# LodgeNetwork - Project Principles

## Notre vision

LodgeNetwork est une plateforme de jeu construite avec les mêmes exigences qu'un projet logiciel professionnel.

Chaque décision est prise pour garantir :

- la stabilité ;
- la sécurité ;
- la maintenabilité ;
- l'évolutivité.

## Nos règles

1. Git est la source de vérité.
2. Toute décision importante est documentée par un ADR.
3. Toute configuration est versionnée.
4. Aucun plugin n'est installé sans validation.
5. Les conventions sont définies avant le développement.
6. Les sauvegardes sont testées régulièrement.
7. Les changements sont reproductibles.
8. La sécurité est prioritaire sur la rapidité.
9. Les composants sont indépendants autant que possible.
10. La simplicité est privilégiée à la complexité.

## Notre objectif

Pouvoir reconstruire l'intégralité de LodgeNetwork uniquement à partir du dépôt Git et de la documentation.
